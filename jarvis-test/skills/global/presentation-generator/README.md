# Скилл: Генератор презентаций

HTML-презентации в браузере: слайды с анимацией, прогрессивным раскрытием, навигацией. Для курсов, питчей, вебинаров, методичек.

## Установка

Положи папку в `~/.claude/skills/presentation-generator/`. После следующего `/reset` скилл активен.

## Как настроить под себя

В папке `context/` лежат 4 файла с шаблонами:

- `speaker.md` — твои данные как спикера
- `audience.md` — твоя целевая аудитория
- `products.md` — твои продукты
- `bonuses.md` — методология бонусов (универсальная, можно не менять)

Заполни три первых файла своими данными. Скилл будет автоматически учитывать их при генерации слайдов — спикер, ЦА, продукт будут твоими.

## Как использовать

Скажи агенту:

```
Сгенерируй презентацию урока на тему «Как настроить рассылку через ИИ»
```

или

```
Сделай слайды для вебинара по теме «Х»
```

Скилл активируется автоматически. Прочитает методологию, шаблоны, контекст и выдаст готовые HTML-слайды в формате `===SLIDE===`.

## Что внутри

- `SKILL.md` — инструкция для Claude (читается автоматически)
- `methodology.md` — правила плотности слайдов, структуры, тона
- `slide-format.md` — спецификация формата `===SLIDE===`
- `components.md` — компоненты слайдов (списки, цитаты, цифры)
- `generators/` — методологии разных типов: урок, вебинар, бонус, воркшоп
- `templates/` — CSS/JS для HTML-презентаций
- `checklists/` — чек-листы проверки качества
- `context/` — твой профиль, ЦА, продукты

## Что НЕ входит

Конвертер в Google Slides (требует личного OAuth-клиента для каждого пользователя) в этом скилле опущен. Можешь работать только с HTML-форматом — этого достаточно для большинства задач.
