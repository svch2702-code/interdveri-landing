## СТРУКТУРА МЕТОДИЧКИ

### Обязательные элементы (в этом порядке)

1. **Header** → `methodology-header` (заголовок + подзаголовок, зелёный градиент)
2. **Табы** → `tabs` (3-5 табов: Основы, Практика, Инструкции, Чеклист, Ресурсы)
3. **Контент табов** → `tab-content` с карточками `cards-grid`
4. **Карточки** → `card` (тема + описание + кнопки действий)
5. **Промпты** → `prompt-block` с кнопкой "📋 Скопировать"
6. **Ссылки на инструменты** → `tool-link` с иконками
7. **Чеклисты** → `step-block` с галочками
8. **Футер** → Meta-дисклеймер + автор + соцсети

**ВАЖНО:** CTA-блоки (бот + канал) используются ТОЛЬКО в лид-магнитах, НЕ в методичках!

## СТРУКТУРА ТАБОВ

### Обязательные табы (3-5 штук):

**1. Основы / Обзор** → Что узнаешь, ключевые концепции, карточки с темами

## ТИПЫ КАРТОЧЕК

### 1. Карточка-тема (с описанием)
```html
<div class="card">
    <div class="card-icon">🎯</div>
    <h3>Название темы</h3>
    <p>Краткое описание темы (2-3 предложения)</p>
    <div class="card-actions">
        <button class="btn btn-primary" onclick="switchTab('practice')">Попробовать →</button>
    </div>
</div>
```

### 2. Карточка с промптом
```html
<div class="card">
    <div class="card-icon">💡</div>
    <h3>Название промпта</h3>
    <p>Что делает этот промпт</p>
    <div class="prompt-block">
        <div class="prompt-header">
            <span class="prompt-label">Промпт</span>
            <button class="btn-copy" onclick="copyPrompt(this)">📋 Скопировать</button>
        </div>
        <div class="prompt-text">Ты — эксперт по {тема}.
Создай {что нужно}.
Формат: {формат}.</div>
    </div>
</div>
```

### 3. Карточка с инструментом
```html
<div class="card">
    <div class="card-icon">🔧</div>
    <h3>Название инструмента</h3>
    <p>Зачем нужен, что делает</p>
    <div class="card-actions">
        <a href="https://example.com" target="_blank" class="btn btn-primary">Открыть инструмент →</a>
    </div>
</div>
```

## ПРОЦЕСС ГЕНЕРАЦИИ МЕТОДИЧКИ

### ШАГ 1: Понимание контекста
- К какой теме относится методичка? (вебинар, урок, воркшоп)
- Какие ключевые концепции/промпты были в материале?
- Какие инструменты упоминались?
- Есть ли скриншоты или демонстрации?

### ШАГ 2: Структура табов
Определи 3-5 табов на основе темы:
- **Всегда:** Основы, Практика, Чеклист
- **Опционально:** Инструкции (если есть пошаговые действия), Ресурсы (если много ссылок)

### ШАГ 3: Создание карточек
Для каждого таба создай 3-6 карточек:
- **Основы:** Карточки с темами и концепциями
- **Практика:** Карточки с промптами
- **Инструкции:** Карточки с шагами (expandable-card)
- **Чеклист:** Список с галочками
- **Ресурсы:** Карточки с инструментами и ссылками

### ШАГ 4: Наполнение контента
- Промпты — готовые к копированию (с плейсхолдерами)
- Инструменты — с реферальными ссылками
- Чеклист — 7-12 пунктов, глаголы в инфинитиве
- Каждая карточка — 1 тема, не больше 3-4 пунктов

### ШАГ 5: Сборка HTML
- Прочитать `templates/methodology.css` → вставить в `<style>`
- Прочитать `templates/methodology.js` → вставить в `<script>`
- Google Fonts: `<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700;800;900&display=swap">`
- Обернуть контент в HTML-структуру
- Футер: дисклеймер + автор + соцсети (из `../config.json`)

### ШАГ 6: Сохранение

**ВАЖНО:** Методичка сохраняется **В ТОЙ ЖЕ ПАПКЕ**, где лежит основной материал!

**Логика:**
```
! Материалы/Для клиентов/[Тип]/
└── ГГГГ-ММ_Название_материала/
    ├── текстовый файл слайдов (.txt)
    ├── презентация (.pptx)
    └── Методичка.html                  ← СЮДА!
```

**Примеры:**
- Вебинар: `Вебинары/2026-02_Промптинг/Методичка.html`
- Воркшоп: `Воркшопы/2026-02_AI_агенты/Методичка.html`
- Урок: `Уроки/2026-02_Google_AI_Studio/Методичка.html`

**Формат имени файла:** ВСЕГДА `Методичка.html` (без дополнительных названий)

**Если пользователь не указал путь:**
- Спроси: "К какому материалу создаём методичку? Укажите путь к папке"
- ИЛИ создай в `! Материалы/Для клиентов/` (если совсем неизвестно)

## ЧЕК-ЛИСТ ПОСЛЕ ГЕНЕРАЦИИ

### Контент
- [ ] Header с заголовком и подзаголовком
- [ ] 3-5 табов (Основы, Практика, Чеклист обязательно)
- [ ] Каждый таб — 3-6 карточек
- [ ] Язык простой (понятен 10-летнему ребёнку)

### Промпты и инструменты
- [ ] Все промпты с кнопкой "📋 Скопировать"
- [ ] Промпты готовы к копированию (с плейсхолдерами)
- [ ] Все инструменты с реферальными ссылками

### Интерактивность
- [ ] Табы переключаются (onclick="switchTab(...)")
- [ ] Кнопки копирования работают (onclick="copyPrompt(this)")
- [ ] Expandable cards раскрываются (onclick="toggleExpand(this)")

### Футер
- [ ] Meta-дисклеймер на месте
- [ ] Instagram* со звёздочкой
- [ ] Подпись автора + соцсети (Instagram*, Telegram, YouTube)

### Техника
- [ ] HTML валидный, все стили inline в `<style>`
- [ ] JavaScript inline в `<script>`
- [ ] Адаптивность (mobile-first)
- [ ] Self-contained файл (не требует внешних зависимостей)
