## ФОРМУЛА ПРОМПТОВ: CRAFT

Все промпты в бонусах создаются по формуле **CRAFT** и оформляются как **шаблоны** — пользователь подставляет свои данные в `[плейсхолдеры]`.

| Буква | Что значит | Пример |
|-------|-----------|--------|
| **C** — Context | Контекст ситуации | "Я хочу создать [тип продукта] для [ниша]" |
| **R** — Role | Роль для нейросети | "Ты — опытный [роль]" |
| **A** — Action | Конкретное действие | "Проанализируй / Составь / Создай..." |
| **F** — Format | Формат результата | "Результат оформи в виде [таблица/список/план]" |
| **T** — Tone | Тон и ограничения | "Простым языком, без жаргона, для новичков" |

**Каждый промпт:**
- Решает одну конкретную задачу
- Содержит `[плейсхолдеры в квадратных скобках]` для подстановки данных
- Оформлен в `prompt-block` с кнопкой "Скопировать"
- Даёт результат за 10-20 минут
- Понятен полному новичку

## ФОРМАТ ИНСТРУКЦИЙ: 1-2-3

Каждый шаг/раздел в бонусе описывается по единой технологии:

```
ШАГ X: [Название шага]

1. ЧТО ДЕЛАЕМ — описание действия простым языком
2. ПРОМПТ — готовый шаблонный промпт по формуле CRAFT (в prompt-block)
3. ЗАЧЕМ — почему этот шаг важен
4. РЕЗУЛЬТАТ — что человек получает (конкретный артефакт)
```

## СТРУКТУРА БОНУСА

### Обязательные элементы (в этом порядке)

1. **Заголовок h1** → `bonus-main-title` (ёмкий, 3-5 слов, с градиентом)
2. **Подзаголовок** → `subtitle` (развёрнутая выгода)
3. **Оглавление** → `bonus-toc` (каждый пункт — оффер!)
4. **hr-разделитель**
5. **Вступление** (150-300 слов) — личная история от лица Дмитрия
6. **5-7 содержательных секций** с h2 + fade-in
   Каждая в формате: ЧТО ДЕЛАЕМ → ПРОМПТ (CRAFT) → ЗАЧЕМ → РЕЗУЛЬТАТ
7. **Резюме** — что узнал, какие шаги прошёл, что умеет
8. **Нативная интрига**: "Один — круто. Но в компании эффективнее."
   > Посыл в духе: "Ты уже можешь сделать [результат]. Но в компании людей, которые в этом разбираются, всегда эффективнее. Когда рядом те, кто уже прошёл этот путь — ты двигаешься быстрее."
   - Это **нативно** подводит к мысли учиться у нас
   - БЕЗ прямой продажи — просто мысль "в компании круче"
9. **CTA-блок** → кнопки на бота и канал
10. **Meta-дисклеймер** + подпись автора + соцсети

## ФОРМАТ ДЛЯ ПЕРЕНОСА В GOOGLE SLIDES (опционально)

Если нужно создать презентационный бонус (слайды вместо HTML):

### Типы слайдов для бонусов:

**Титульный:**
```
===SLIDE===
ТИП: титульный
НОМЕР_УРОКА: Бонус
ЗАГОЛОВОК: 50 готовых промптов для бизнеса
```

**Важно про НОМЕР_УРОКА:**
- Не только "Бонус" — может быть "День 2", "Блок 3: Практика", "Часть 2", "Тема: Продвинутый уровень"
- Может появляться в ЛЮБОМ месте презентации (не только в начале)
- Используется для разделения тем/блоков внутри презентации

**Контентный (список промптов):**
```
===SLIDE===
ТИП: контент
ЗАГОЛОВОК: Промпты для маркетинга
ТЕКСТ:
• Генерация идей для постов
• Создание email-рассылок
• Анализ целевой аудитории
```

**Промпт:**
```
===SLIDE===
ТИП: промпт
ЗАГОЛОВОК: Промпт 1: Генерация идей для контента
ПРОМПТ:
Ты — эксперт по контент-маркетингу...
```

### ВАЖНЫЕ ПРАВИЛА для слайдов бонусов:

1. **СПИКЕР** — ВСЕГДА в заметки, НЕ на слайд!
2. **ТЕКСТ** — короткие буллеты (3-5 пунктов, каждый 1-2 строки)
   - **КРИТИЧНО: Простой язык БЕЗ жаргона!**
   - Разговорный, но деловой стиль
   - Понятно 10-летнему ребёнку
   - БЕЗ канцелярита (не "в рамках", "посредством")
   - БЕЗ жаргона постов ("короче", "классный", "крутой")
   - Примеры:
     - ✅ "Открываем AI Studio и создаём первый промпт"
     - ✅ "Результат готов за 10 минут — рабочий чат-бот"
     - ✅ "Вы создаёте продукт БЕЗ кода — только промпты"
     - ❌ "Использование инструмента AI Studio" (канцелярит)
     - ❌ "Короче, делаем так..." (жаргон — только для постов!)
3. **ПРОМПТ** — готовый к копированию (copy-paste ready)
4. **Все поля** — ЗАГЛАВНЫМИ БУКВАМИ с двоеточием
