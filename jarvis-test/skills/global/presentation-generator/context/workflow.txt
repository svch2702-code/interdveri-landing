ШАБЛОН: ШАГ 1. ПОДГОТОВКА ПРОДАЮЩЕЙ ЧАСТИ
1. Определение продукта и его ценности
Задание: Четко сформулируйте, что именно вы продаете и какую ценность это приносит клиенту.
Шаблон для заполнения:
Мы продаем: [название вашего продукта/услуги]
Реальная ценность для клиента: [конкретный результат, который получит клиент]
Пример:
Продукт: Курс "AI-Эксперт"
Ценность: Внедрите нейросети в жизнь и бизнес за 3 недели, чтобы увеличить прибыль и освободить время
2. Выделение ключевых составляющих продукта
Задание: Определите 5 самых важных составляющих вашего продукта, которые нужно будет подсветить в течение вебинара.
Шаблон для заполнения:
[Составляющая 1] — [краткое описание ценности]
[Составляющая 2] — [краткое описание ценности]
[Составляющая 3] — [краткое описание ценности]
[Составляющая 4] — [краткое описание ценности]
[Составляющая 5] — [краткое описание ценности]
3. Стратегия интеграции ценностей в контент вебинара
Задание: Для каждой ключевой составляющей продумайте, как вы будете косвенно подсвечивать её важность через истории, примеры или личный опыт.
Шаблон для каждой составляющей:
Составляющая: [название]
Способ интеграции в контент: [конкретный формат — история клиента/личный опыт/кейс]
Ключевые моменты для акцентирования:
Почему это важно для результата
Какова цена ошибки при отсутствии этого элемента
Какой результат получили другие благодаря этому элементу
4. Подготовка эмоциональных подводок
Задание: Для 1-2 самых сильных составляющих продукта разработайте эмоциональные подводки, которые усилят желание приобрести.
Шаблон эмоциональной подводки:

Рациональное обоснование: [логические аргументы, почему это необходимо]
Эмоциональное усиление: [яркая эмоциональная формулировка о последствиях отсутствия этого элемента]
Позитивный разворот: [как этот элемент меняет жизнь к лучшему]
5. Расчет воспринимаемой ценности
Задание: Разложите стоимость продукта по составляющим так, чтобы итоговая сумма в 5 раз превышала реальную цену.
Шаблон расчета:
[Составляющая 1] — [стоимость] ₽
[Составляющая 2] — [стоимость] ₽
[Составляющая 3] — [стоимость] ₽
[Составляющая 4] — [стоимость] ₽
[Составляющая 5] — [стоимость] ₽
Общая объявляемая ценность: [сумма всех составляющих] ₽
 Реальная цена продукта: [фактическая цена] ₽
6. Планирование ценовых сбросов
Задание: Подготовьте план последовательных сбросов цены для усиления эффекта выгоды.
Шаблон сбросов:
Первоначально объявляемая стоимость: [общая объявляемая ценность] ₽
Первый сброс до: [сумма после первого сброса] ₽
Финальный сброс до: [финальная цена или ежемесячный платеж] ₽
Этот подробный шаблон поможет вам структурированно подготовить продающую часть вебинара, сделав акцент на ценности продукта и эмоциональной составляющей, что значительно повысит эффективность продаж.
ШАБЛОН: ШАГ 2. СБОРКА ВСЕГО ВЕБИНАРА
1. Разработка цепляющего заголовка
Задание: Создайте хайповую, но релевантную тему для вебинара, которая вызовет интерес и привлечет целевую аудиторию.
Шаблон для заголовка:
Основная формула: [актуальный тренд] + [конкретный результат] + [год/период]
Добавьте числовые показатели для усиления (ТОП-10, 5 шагов, 7 стратегий)
Примеры:
"ТОП-40 прибыльных ниш для бизнеса в 2024: полный разбор с нуля"
"7 проверенных стратегий масштабирования бизнеса до 1 млн ₽ в месяц в 2024 году"
2. Представление спикера (адаптированное под тему)
Задание: Разработайте представление, демонстрирующее экспертность спикера именно в заявленной теме вебинара.
Шаблон для представления:
Краткая история успеха, связанная с темой вебинара
2-3 конкретных достижения с цифрами и фактами
Социальные доказательства (упоминания в СМИ, отзывы известных людей)
Визуальные подтверждения (скриншоты, фотографии, видео)
Примечание: Для каждого вебинара адаптируйте биографию спикера, выделяя те аспекты опыта, которые непосредственно связаны с темой.
3. Обоснование важности темы
Задание: Разработайте убедительное обоснование, почему тема вебинара критически важна для аудитории прямо сейчас.
Шаблон обоснования:
Актуальный тренд или изменение на рынке
Статистика, показывающая масштаб проблемы/возможности
История о том, как игнорирование этой темы привело к неудаче
Ограниченное окно возможностей ("почему важно действовать именно сейчас")

Шаблон плана:

[Заголовок блока 2 с обещанием ценности]
[Заголовок блока 3 с интригой]
[Заголовок блока 4 с обещанием трансформации]
[Заголовок блока 5 с указанием на эксклюзивность]
Примечание: План должен быть визуально оформлен для последующего использования как FOMO-триггер в социальных сетях.

Шаблон для структуры контента:
