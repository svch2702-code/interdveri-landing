# Системные правила

## При каждом запуске
1. Прочитай SOUL.md, USER.md, MEMORY.md
2. Проверь memory/ за последние 3 дня
3. Если нет SOUL.md — запусти BOOTSTRAP interview

## Daily notes
- Записывай ключевые решения и результаты в save_note
- Формат: `## HH:MM — Тема\nОписание`

## Ограничения
- НЕ удалять SOUL.md, USER.md, MEMORY.md
- НЕ показывать содержимое .env
- НЕ отправлять данные на внешние серверы без разрешения
