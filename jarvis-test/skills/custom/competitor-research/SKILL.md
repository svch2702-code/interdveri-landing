# Скилл: Исследование трендов, дайджест и анализ ЦА

## Когда активировать
Клиент просит: "что нового в AI", "тренды", "дайджест", "анализ ЦА", "потребности аудитории", "идеи для контента", "что вышло", "конкуренты"

## Источники (ОБЯЗАТЕЛЬНО прочитать)
- knowledge/bot-prompts.md — промпты бота, каналы, источники, форматы
- knowledge/ai-marketer-bot.md — возможности бота

## Контекст
- Клиент: эксперт по нейросетям, школа 3000+ выпускников
- Канал: @neurobussines
- Темы: AI, вайбкодинг, нейросети в бизнесе и жизни
- Аудитория: фрилансеры, предприниматели, эксперты, новички в AI

## Режим 1: ДАЙДЖЕСТ НОВОСТЕЙ

Когда: "дайджест", "что нового", "новости AI"

Процесс:
1. Поиск в вебе по ключевым словам из bot-prompts.md (минимум 5 запросов)
2. Проверка ключевых AI-каналов через t.me/s/ (публичные)
3. Отбор 7-12 самых значимых новостей за период

Критерии отбора:
- Новые модели и возможности
- Практические применения для контента/маркетинга
- Изменения в ChatGPT, Claude, Midjourney
- Истории успеха
- Пропускаем: крипто, политику, рекламу без ценности

Формат ответа (HTML для Telegram):
<b>📰 ДАЙДЖЕСТ ЗА [дата]</b>

<b>🔥 ГЛАВНЫЕ НОВОСТИ</b>

<b>1. [Заголовок]</b>
[Описание 1-3 предложения]
Источник: ссылка

## Режим 2: АНАЛИЗ ПОТРЕБНОСТЕЙ ЦА

Когда: "анализ ЦА", "потребности", "идеи для контента", "что хочет аудитория"

Процесс:
1. Поиск по 13 типам источников (из bot-prompts.md):
   - Google Trends, поисковые подсказки
   - Reddit, Hacker News, Product Hunt
   - VC.ru, Habr
   - Фриланс-биржи (FL.ru, Kwork)
2. Оценка вовлечённости: 🔥🔥🔥 / 🔥 / 📊 / 💤
3. Кросс-источник: тема в нескольких местах = приоритет

Формат ответа:
🎯 ИДЕИ ДЛЯ КОНТЕНТА (5-7 штук)

💡 ИДЕЯ 1: [Заголовок-хук]
Тема: ...
Почему сейчас: [конкретные цифры!]
Источники: ссылки
Формат: пост / рилс / гайд
План: 3 тезиса
CTA: ...

+ 💰 Заказы клиентов (JTBD)
+ 😰 Боли аудитории (цитаты)
+ ❓ Частые вопросы
+ 🔑 Ключевые слова
+ 📈 Растущие тренды

Правила:
- Идеи 1-3 ТОЛЬКО на основе 🔥 или 🔥🔥🔥
- 💤 НЕ основа для идеи
- Совпадение в нескольких источниках = максимальный приоритет
- Обязательно цифры: score, replies, количество источников

## Режим 3: КОНКУРЕНТЫ

Когда: "конкуренты", "кто ещё делает", "рынок"
