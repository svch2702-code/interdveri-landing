# .claude/skills/ — команда специалистов Вашего Агента

Эта папка — для скиллов. Скилл = инструкция для одного специалиста, к которому Агент будет обращаться сам, когда увидит подходящую задачу.

> Скиллы лежат **в проекте**, а не глобально — чтобы при переезде Агента на сервер вся команда уехала вместе с ним.

---

## Что уже установлено

4 скилла идут в комплекте с шаблоном и готовы к работе сразу:

- `discovery-interview/` — сбор ТЗ через интервью
- `content-creator/` — написание контента
- `fullstack-developer/` — разработка кода
- `frontend-design/` — дизайн интерфейсов

Ниже — описание каждого и инструкции для обновления до последней версии от автора.

---

## Описание скиллов и обновление

### 1. Интервьюер — собирает ТЗ через семифазное интервью

**Скилл:** [`discovery-interview`](https://github.com/parcadei/Continuous-Claude-v3) (автор: Parcadei)

**Что делает:** превращает размытую идею заказчика в детальную спецификацию через 25 структурированных вопросов с вариантами ответа.

**Фраза для Claude Code:**

```
Поставь мне скилл discovery-interview от parcadei.
Адрес: https://github.com/parcadei/Continuous-Claude-v3
Положи в папку .claude/skills моего проекта jarvis.
```

---

### 2. Писатель — пишет посты, статьи, рассылки

**Скилл:** [`content-creator`](https://github.com/Shubhamsaboo/awesome-llm-apps) (автор: Shubham Saboo)

**Что делает:** пишет тексты по форматам — пост в соцсети, блог-пост, email-рассылка, описание продукта. Использует Ваш голос из `SOUL.md` + правильную структуру под формат.

**Фраза для Claude Code:**

```
Поставь мне скилл content-creator из github.com/Shubhamsaboo/awesome-llm-apps.
Положи в папку .claude/skills моего проекта jarvis.
```

---

### 3. Разработчик — пишет код для прототипов и сайтов

**Скилл:** [`fullstack-developer`](https://github.com/Shubhamsaboo/awesome-llm-apps) (автор: Shubham Saboo)

**Что делает:** пишет полнофункциональный код по современным стандартам. Прототип лендинга, скрипт автоматизации, Telegram-бот, интеграция «заявка с сайта → запись в таблицу». Вам не нужно разбираться в технологиях — Агент сам выбирает что подходит под задачу.

**Фраза для Claude Code:**

```
Поставь мне скилл fullstack-developer из github.com/Shubhamsaboo/awesome-llm-apps.
Положи в папку .claude/skills моего проекта jarvis.
```

> Если ставите Писателя и Разработчика по очереди — это нормально, оба из одного репозитория.

---

### 4. Дизайнер (бонус) — для тех, кому нужны лендинги

**Скилл:** [`frontend-design`](https://github.com/anthropics/claude-code) (официальный, от Anthropic)

**Что делает:** делает лендинги и интерфейсы, которые не выглядят «как сделано на ИИ». Сначала выбирает стиль (минимализм, ретро, современный), потом подбирает шрифты и цвета, потом пишет код.

**Фраза для Claude Code:**

```
Поставь мне скилл frontend-design от Anthropic.
Адрес: https://github.com/anthropics/claude-code
Положи в папку .claude/skills моего проекта jarvis.
```

---

### 5. Superpowers — плагин-методолог (не скилл, ставится иначе)

**Плагин:** [`superpowers`](https://github.com/obra/superpowers) (автор: Jesse Vincent / obra)

**Что делает:** ведёт сложные задачи по этапам через три горячие команды:

- `/brainstorm` — мозговой штурм идеи до начала работы
- `/write-plan` — пошаговый план реализации
- `/execute-plan` — автоматическое исполнение плана с самопроверкой

**Команда установки** (пишется прямо в окно Claude Code, не сюда — это slash-команда):

```
/plugin install superpowers@claude-plugins-official
```

> Superpowers ставится **глобально** в `~/.claude/plugins/`, не в эту папку. Будет работать во всех Ваших проектах.

## Подсказка Агенту после установки

Когда все скиллы поставлены — попросите Агента дописать «табличку на двери офиса»:

```
Допиши в CLAUDE.md блок «Моя команда» со списком установленных скиллов
и плагина Superpowers. И в MEMORY.md добавь запись об установке
с сегодняшней датой.
```

Технически Claude Code видит скиллы в этой папке автоматически. Но в спорных случаях он может не догадаться, что у него есть подходящий специалист — табличка в `CLAUDE.md` помогает Агенту вспомнить о команде и активнее её использовать.

## Не забудьте зафиксировать в Git

После установки всех скиллов:

```
Закоммить и запушь .claude/skills/ с сообщением
«skills: установил команду специалистов»
```

Так команда уедет вместе с Агентом, если Вы потом перенесёте его на сервер.

---

## Если Вы уже скачали шаблон без папки .claude/skills/

Если Вы прошли Урок 4 раньше, чем эта папка появилась в шаблоне — скачайте только её, не пересоздавая весь Агент:

**Через окно Claude Code (рекомендуется):**

```
Скачай папку .claude/skills из репозитория Ntmib/jarvis-architect
в мой проект jarvis. Адрес папки на GitHub:
https://github.com/Ntmib/jarvis-architect/tree/main/.claude/skills
```

**Или вручную через terminal** (для продвинутых):

```bash
npx degit Ntmib/jarvis-architect/.claude/skills .claude/skills
```

После этого продолжите установку скиллов по фразам из этого файла.
