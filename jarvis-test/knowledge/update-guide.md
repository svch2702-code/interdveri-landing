# Как обновить бота до версии 2.0

Эта инструкция для тех, кто уже установил бота ранее и хочет обновить его до новой версии.

Обновление занимает 2 минуты. Все ваши данные (SOUL.md, MEMORY.md, настройки) сохранятся — обновляется только код бота.

---

## Что нового в версии 2.0

- Бот теперь запоминает контекст разговора (семантическая память)
- Появилось меню настроек (/settings) — подключение API-ключей прямо из Telegram
- Бот показывает анимацию «думаю...» пока обрабатывает запрос
- Можно отправлять ссылки на Google Docs/Sheets — бот сам загрузит содержимое
- Расписание задач — бот напомнит о важном
- Новая команда /update — обновление в одно нажатие (для будущих версий)

---

## Пошаговая инструкция

### Шаг 1. Откройте VS Code

Откройте программу VS Code на вашем компьютере. Обычная программа, не в браузере.

### Шаг 2. Подключитесь к серверу

1. На левой панели VS Code найдите иконку в виде монитора с маленьким значком (Remote Explorer). Если не видите — нажмите Ctrl+Shift+P, напишите "Remote Explorer" и выберите его
2. В открывшейся панели найдите раздел Tunnels — там будет название вашего сервера
3. Нажмите на него — VS Code подключится к серверу

### Шаг 3. Откройте терминал

Нажмите на клавиатуре Ctrl и ` (кнопка с обратной кавычкой, обычно слева от цифры 1, там же где буква Ё).

Внизу экрана откроется панель терминала — чёрное окно с мигающим курсором.

### Шаг 4. Вставьте команду обновления

Скопируйте эту команду и вставьте в терминал:

```
cd /home/agent/.agent/bot && curl -fsSL https://raw.githubusercontent.com/Ntmib/jarvis-architect/main/bot/update-bot.sh | bash
```

Как вставить: нажмите правой кнопкой мыши в терминале → Paste. Или Ctrl+Shift+V.

После вставки нажмите Enter.

### Шаг 5. Дождитесь завершения

Обновление займёт 30-60 секунд. Вы увидите прогресс:

```
=== Agent Bot Update ===
[1/6] Downloading files...
[2/6] Verifying critical files...
[3/6] Syntax check...
[4/6] Backing up current files...
[5/6] Installing update...
[6/6] Restarting bot...

=== Update complete! Version: 2.0.0 ===
```

Если видите "Update complete!" — всё готово. Бот обновлён и уже работает.

### Шаг 6. Проверьте

Напишите боту в Telegram команду /status — он покажет новую версию.

---

## Если что-то пошло не так

**"Bot directory not found"** — скрипт не нашёл папку бота. Убедитесь, что вы подключены к правильному серверу через VS Code.

**"Syntax error in new index.js"** — обновление отменено автоматически, бот работает на старой версии. Напишите в поддержку.

**Бот не отвечает после обновления** — скрипт автоматически откатит изменения и восстановит старую версию. Если не помогло — напишите в поддержку.
